
# ODBC_v06

*A fork of the original Julia library for interacting with the ODBC API within Julia 0.6.4.*

## Installation

```julia
julia> include("ODBC_v06.jl")
```

## Project Status

The package is tested against Julia `0.6.4` on Windows.
